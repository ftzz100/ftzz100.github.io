iphone照片管理

Posted on 2020-05-11 In Mac

手机上的照片很多了，快一年没备份了。相册也占了70G的空间，手机也天天弹框说空间不足，所以得整理一下手机相册了。

Start

设备
iphone7 13.2.3
macOS 10.14.6

前提
不考虑使用iCloud，因为iCloud只有5G，照片视频有70G，升级扩容没必要。

快一年没有备份手机照片了，这次是新电脑系统备份，上次备份还是旧系统，感觉差不多。

连接手机

手机电脑通过usb连接之后，打开电脑的照片软件，左边侧边栏出现设备时，选中自己的手机设备，一开始一片空白，可能是图片太多了，要加载一会儿才可以预览到手机里的照片缩略图，不过不能打开放大。

导入原始格式照片

选中一张照片，右上角有“导入后删除项目”，以前没用过，现在就测试一下。导入所选的一张照片到电脑，果然导完后照片就删除了，在手机里也删除了，手机回收站里也没有。这时候选中侧边栏的“导入”，可以看到刚才导入的照片，选中照片用空格键放大查看，发现左上角有实况的标示。将鼠标移动到这标示上面，会显示live效果。

这时候去相册包内容里查看照片，只有jpg格式的。记得以前一次导入大量多张live照片，会同时生成一张图片，一个视频的文件。可是有时候导入就没有，要等到下一次导入照片才会将视频文件备份到包内容里，这点有些奇怪。

电脑保存Live照片

总之现在只有照片，或许下次再导入照片的时候才会出现这次对应的视频文件，如果现在我想立刻得到所对应的视频呢？没问题，在照片软件的“导入”里，选中刚才那张导入的照片，然后在上面工具栏，选择文件-> 导出 ->导出一张未修改的原件

这里没有勾选“将 IPTC 导出为 XMP”，直接点导出。如果勾选，会在导出的照片旁边放一个XMP文件，这个文件包含了照片的一些原始信息。

然后选择导出的目录，导出后可看到导出了一个静止图像文件和一个视频文件。实况照片得以保存

手机传入实况照片

live照片在电脑里是得到保存了，可是如果想传回手机也保留live photo效果呢。于是测试了一下，将这对应的图片与视频同时选中，然后通过隔空传递传到手机。然后打开手机相册，可以看到多了一个文件，就是实况照片。看来手机会自动将两个文件自动合并起来。当然如果手机实况照片隔空传递到电脑，就只是传过去静态图像了。
